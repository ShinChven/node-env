# node-env

> Node docker images based on ubuntu

## version

- node: 16.x
- ubuntu: latest

## node 16 Images

- `shinchven/node:16` - The base image
- `shinchven/node:16-build` - Build and compile
- `shinchven/node:16-deployment` - Deployment env with pm2-docker and typescript
